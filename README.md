## üöÄ How to Use

### **Method 1: The Easy Way (Extension)**
1.  **Install** [Grok CleanPro from the Firefox Add-on Store](https://addons.mozilla.org/en-US/firefox/addon/grok-cleanpro/).
2.  Go to your [Grok Deleted Conversations](https://grok.com/deleted-conversations) page.
3.  Click the extension icon and hit **Start Cleaning**!

### **Method 2: The Script Way (Console)**
1.  Open the **`script.js`** file in this repository and copy the code.
2.  Navigate to [Grok Deleted Conversations](https://grok.com/deleted-conversations).
3.  Open the Browser Console:
    * **Win/Linux:** `F12` or `Ctrl + Shift + J`
    * **macOS:** `Cmd + Option + J`
4.  Paste the code and hit **Enter**. ‚ö°Ô∏è
